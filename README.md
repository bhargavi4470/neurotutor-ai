# 🧠 NeuroTutor AI

NeuroTutor AI is an advanced, AI-powered learning companion designed to personalize education. Leveraging the power of Google's Gemini AI, it provides interactive tutoring, explains complex concepts, and tracks student progress with dynamic data visualizations.

## ✨ Features

- **Personalized AI Tutoring**: Real-time interaction with Gemini AI for instant help and explanations.
- **Progress Tracking**: Interactive charts and analytics built with Recharts to monitor learning milestones.
- **Modern UI/UX**: Sleek, responsive interface designed for an optimal learning experience with Lucide icons.
- **Fast Development**: Powered by Vite and React for a lightning-fast user experience.

## 🚀 Tech Stack

- **Frontend**: React 19, Vite
- **AI Integration**: @google/genai (Gemini AI)
- **Data Visualization**: Recharts
- **Icons**: Lucide React
- **Language**: TypeScript

## 🛠️ Run Locally

**Prerequisites:**  Node.js

1. **Clone the repository** (after uploading to GitHub):
   ```bash
   git clone https://github.com/bhargavi4470/neurotutor-ai.git
   cd neurotutor-ai
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up environment variables**:
   Create a `.env.local` file and add your Gemini API key:
   ```env
   GEMINI_API_KEY="your_api_key_here"
   ```

4. **Start the development server**:
   ```bash
   npm run dev
   ```
